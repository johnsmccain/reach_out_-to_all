                              STEP-BY-STEP GUIDE
================================================================================

STEP 1: OPEN SUPABASE SQL EDITOR
────────────────────────────────────────────────────────────────────────────
  1. Open your browser
  2. Go to: https://supabase.com/dashboard/project/xpklyizyeihcjlejgylu/sql
  3. Click "New Query" or use the existing editor

STEP 2: COPY THE SETUP SCRIPT
────────────────────────────────────────────────────────────────────────────
  1. In your code editor, open: setup-supabase.sql
  2. Select all text (Ctrl+A or Cmd+A)
  3. Copy (Ctrl+C or Cmd+C)

STEP 3: PASTE AND RUN IN SUPABASE
────────────────────────────────────────────────────────────────────────────
  1. In the Supabase SQL Editor, paste the script (Ctrl+V or Cmd+V)
  2. Click the green "RUN" button (or press Ctrl+Enter / Cmd+Enter)
  3. Wait 5-10 seconds
  4. You should see: "Success. No rows returned"

STEP 4: VERIFY IT WORKED
────────────────────────────────────────────────────────────────────────────
  1. In Supabase, click "Table Editor" in the left sidebar
  2. You should see 11 tables:

     ✓ prayer_requests
     ✓ volunteers
     ✓ soul_count
     ✓ sermons
     ✓ documents
     ✓ statistics
     ✓ articles
     ✓ article_reactions
     ✓ article_comments
     ✓ daily_quotes

STEP 5: REFRESH YOUR APPLICATION
────────────────────────────────────────────────────────────────────────────
  1. Go to your app: http://localhost:5173
  2. Refresh the page (F5 or Ctrl+R / Cmd+R)
  3. Errors should be gone! ✅

================================================================================
                           TROUBLESHOOTING
================================================================================

IF YOU SEE SQL ERRORS:
  → Make sure you copied the ENTIRE setup-supabase.sql file
  → Try running the script again (it's safe to run multiple times)

IF TABLES DON'T APPEAR:
  → Refresh the Table Editor page
  → Check that you're in the correct project (xpklyizyeihcjlejgylu)

IF ERRORS PERSIST AFTER SETUP:
  → Check browser console (F12) for specific error messages
  → Verify .env file has correct Supabase credentials
  → Make sure your app is running (npm run dev)
